1. Clone the repo
2. Install the global requirements: npm install -g gulp bower browserify
3. Install json-server (https://github.com/typicode/json-server) globally: npm install -g json-server
4. Install the local requirements: npm install
5. Install the Bower components: bower install
6. Launch json-server: json-server --watch db.json
7. Run app locally: gulp connect

App will be served on http://localhost:8888
